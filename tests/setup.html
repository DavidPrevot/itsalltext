Purpose: to verify that a new version of IAT! has been setup correctly.

Use this textarea for the tests below:

  1. When you mouse over the textarea an edit button should appear.
  2. If you right click on the textarea there should be a context menu item saying "It's All Text!"
  3. (Non-OSX platforms only) If you have never installed IAT! for this browser
    1. Click the edit button. You should get a dialog that says that it was unable to open your editor.
    2. Click on the preferences button.
    3. Set the editor.
    4. Close the preferences. The editor you chose should pop up and should contain the text from the textarea.
