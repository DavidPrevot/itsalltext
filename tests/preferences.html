Purpose: to verify that the preferences exist are work correctly.

Use this textarea for the tests below:

  1. Go to the menu Tools -> It's All Text! -> Preferences
  2. Leaving the preferences pane open (if possible on your OS) do the following for each of the settings for Edit button positions including disable:
    1. Select the setting.
    2. Mouse over the textarea. You should see the edit button in the correct place (or nowhere in the case of 'disable').
  3. Set the hot key to something (such as alt-e).
  4. Click in the textarea.
  5. Press the hot key. An editor should open up.
  6. Close the editor.
  7. Unset the hot key by pressing Escape, Backspace or Delete.
